# 🏧 ATM Machine System (Python)

## 🔹 Overview

This is a Command-Line Interface (CLI) based ATM Machine System built using Python.
It allows users to perform basic banking operations like checking balance, depositing money, and withdrawing money.

---

## 🔹 Features

* 💰 Check account balance
* ➕ Deposit money
* ➖ Withdraw money
* ⚠️ Handles insufficient balance
* 🔒 Input validation for safe operations

---

## 🔹 Technologies Used

* Python

---

## 🔹 How It Works

* The system maintains a variable `balance` to store the current account balance
* Users interact through a menu-driven interface
* Based on user input, operations like deposit, withdraw, or display balance are performed

---

## 🔹 How to Run

1. Clone the repository:
   git clone https://github.com/your-username/ATM-System.git

2. Navigate to the folder:
   cd ATM-System

3. Run the program:
   python atm.py
